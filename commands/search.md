# yongle-search

检索永乐大典知识库中的历史经验和决策。

## Arguments

- `query` (required): 搜索关键词，将匹配标题、摘要和标签。
- `--tags:<tag>` (optional): 按标签过滤。
- `--type:<type>` (optional): 按解决类型过滤 (如: Bug, Implementation, Decision)。
